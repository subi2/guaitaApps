Manual d'ús

Escriu i edita Markdown com un processador de textos.

Guaita .md Editor és un editor complet de Markdown en una sola pàgina: escriu directament a la vista formatada o al codi, tot dos sincronitzats; insereix format, taules i llistes des dels menús; cerca i substitueix, i exporta a HTML, LaTeX o PDF.

01Introducció

Un editor de Markdown en una sola pàgina

Guaita .md Editor és una aplicació autònoma: un únic fitxer HTML que s'obre a qualsevol navegador modern. Et deixa crear i modificar documents Markdown amb una barra de menús completa (Fitxer, Edita, Insereix, Format, Taula, Visualitza), treballant alhora sobre la vista formatada (com un processador de textos) i el codi markdown.

Tot el processament passa dins del teu navegador. Cap document no s'envia a cap servidor: pots treballar sense connexió i amb material privat.

Edició dual

02Com començar

Crea o obre un document
  1. Des de la pantalla inicial, prem Nou document per començar en blanc, o Obre fitxers… per carregar-ne un existent.
  2. També pots fer Fitxer → Nou document (Ctrl+N) o Fitxer → Obre… (Ctrl+O) des de la barra de menús.
  3. Arrossega i deixa anar fitxers .md sobre la finestra per obrir-los; cadascun ocupa una pestanya.
  4. Comença a escriure a la vista formatada o al codi: se sincronitzen soles.
Enganxa amb format
Si enganxes text amb format (per exemple des d'una pàgina web o un document), l'editor el converteix automàticament a markdown.

04Les dues vistes

Formatat i codi, sempre en sincronia

Al menú Visualitza (o amb els botons de la barra) pots activar o desactivar cada panell:

  • Vista formatada — escrius directament sobre el text amb estil, com en un processador de textos.
  • Vista de codi (Ctrl+E) — edites el markdown en brut.
  • Esquema (índex) — panell lateral amb els títols per navegar-hi.
  • Tema clar / fosc — canvia l'aparença de tota la interfície.

Quan tens les dues vistes obertes alhora, arrossega el separador central per repartir l'amplada. Qualsevol canvi en una vista es reflecteix a l'instant a l'altra.

07Taules i CSV

Crea i manté taules sense esforç

El menú Taula reuneix totes les eines per treballar amb taules Markdown:

  • Genera taula… — crea una taula buida amb les files i columnes que indiquis.
  • Enganxa CSV / TSV com a taula… (Ctrl+Maj+V) — enganxa dades separades per comes o tabuladors i les converteix en taula, amb previsualització editable.
  • Formata i alinea columnes — endreça la taula sota el cursor perquè les vores quedin alineades.
  • Afegeix / elimina fila i columna — sobre la taula on tens el cursor.
  • Copia taula com a CSV / TSV — treu les dades cap a un full de càlcul.

08Cerca i substitueix

Troba i canvia text

Obre la barra amb Edita → Cerca i substitueix… o Ctrl+F. Escriu el text a cercar i, opcionalment, el text de substitució. Fes servir Següent per anar saltant coincidències, Substitueix per canviar-ne una i Substitueix-ho tot per canviar-les totes. La casella Maj/min distingeix majúscules i minúscules, i el comptador indica quantes coincidències hi ha.

11Resolució de problemes

Si alguna cosa no va
Les dues vistes no coincideixen
La vista formatada i el codi es sincronitzen automàticament. Si algun cop sembla que no, fes un petit canvi en qualsevol de les dues o alterna-les des del menú Visualitza per refrescar.
Una drecera de línia no fa res
Les eines de línia (duplicar, moure, ordenar…) actuen sobre la vista de codi. Assegura't que el cursor és a l'editor de codi i que hi ha una línia o selecció activa.
La taula queda desalineada
Col·loca el cursor dins la taula i fes Taula → Formata i alinea columnes. La visualització correcta es veu sempre a la vista formatada, encara que el codi tingui les vores irregulars.
Perdo els canvis en recarregar
Els documents viuen en memòria dins la sessió del navegador. Desa sovint amb Ctrl+S; en recarregar la pàgina, els documents no desats s'han de tornar a obrir.
Vull només llegir, no editar
Per a la simple lectura de fitxers Markdown existents tens l'aplicació germana Guaita .md, un visor lleuger amb pestanyes i índex.
