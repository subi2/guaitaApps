01Introducció

Un visor i analitzador d'IFC en una pàgina

Guaita Query és una eina autònoma: un únic fitxer HTML que s'obre a qualsevol navegador modern. Llegeix fitxers IFC2x3 i IFC4 i els representa en 3D, amb l'arbre espacial (edifici, plantes, espais) i l'arbre de classes, les propietats i les quantitats de cada element.

Tot el processament es fa localment al teu navegador; cap fitxer s'envia enlloc.

3D real

Model IFC navegable amb òrbita, zoom i vistes.

Estructura i dades

Arbre espacial, classes, propietats i quantitats.

Consulta i auditoria

Filtra elements i revisa la qualitat del model.

Notes en BCF

Anotacions amb captura, exportables a BCF 2.1.

02Obrir un IFC

Tres maneres de començar
  1. Arrossega i deixa anar el fitxer .ifc sobre el visor.
  2. Fes clic a la zona buida («Arrossega un fitxer .ifc aquí») per obrir el selector.
  3. Prem O o vés a Fitxer → Obre IFC… des de la barra de menús.

Mentre es carrega veuràs un indicador. Quan acaba, el model apareix enquadrat, els panells s'omplen amb l'estructura i les classes, i la barra d'estat mostra les dades del fitxer.

05Estructura i classes

Recorre el model per arbres

La pestanya Estructura mostra l'arbre espacial del model (projecte → edifici → plantes → espais i elements), tal com està organitzat a l'IFC. La pestanya Classes agrupa els elements per classe IFC (murs, portes, finestres, forjats…). Fes clic a qualsevol node per seleccionar-lo al visor i veure'n les propietats a la dreta.

Al panell dret trobaràs les propietats (Psets) i les quantitats (longituds, àrees, volums, recomptes) de l'element seleccionat, i pots copiar-ne el GUID des del menú Edita.

07Taula d'elements i consulta

Els elements en forma de full

Amb D (o Visualitza → Taula d'elements) obres un dock inferior amb tots els elements en files. Pots afegir columnes de propietats, i clicar una fila per seleccionar l'element al visor. El botó Exporta CSV desa la taula per obrir-la a Excel.

Per a filtres avançats, les vistes intel·ligents (pestanya Vistes) construeixen conjunts d'elements a partir de regles sobre classes, propietats, materials, plantes o el nom, i els poden agrupar i pintar amb mapes de color.

08Mesura i seccions

Amida i talla el model

Mesura

Activa-la amb M o Eines → Mesura distàncies. Fes clic a dos punts per obtenir-ne la distància; Esc surt del mode.

Caixa de secció

Eines → Caixa de secció activa una caixa que retalla el model per plans, per mirar-ne l'interior sense que la geometria del davant t'ho tapi.

09Vistes i notes (BCF)

Desa punts de vista i comparteix incidències

Punts de vista

Eines → Captura punt de vista desa la càmera i la selecció actuals a la pestanya Vistes, amb una miniatura. Les vistes intel·ligents es poden exportar en format BIMCollab (JSON), per grups o individualment.

Notes i BCF

A la pestanya Notes, Nova nota captura una imatge, la càmera i els elements seleccionats. Les notes s'exporten en BCF 2.1 (compatible amb Revit, Navisworks i BIMcollab) o en un informe HTML autònom.

10Auditoria i comparació

Revisa la qualitat i compara versions

Auditoria del model

Eines → Auditoria del model comprova regles de qualitat (propietats informades, materials assignats, elements en plantes, GUID únics, mínims per classe…) i genera un informe que pots exportar en CSV o HTML. Pots carregar normes BEP/EIR des del menú Fitxer per validar contra els teus requisits.

Comparació de versions

Fitxer → Compara amb una versió… carrega un segon IFC i en ressalta les diferències respecte del model obert (elements afegits, eliminats o modificats).

Solar i volumetria
Els menús Solar i Volumetria permeten carregar un solar (GeoJSON/GML/LandXML), encaixar-lo amb el model i generar o exportar volumetries urbanístiques (gàlibs) a IFC o DXF.

11La Guia

Pregunta sobre el model

La pestanya Guia (també Eines → Guia · Consulta el model…) és un copilot IFC. En mode local respon amb dades del model (recomptes, classes, propietats, resum). Escriu la pregunta i prem Enter.

Si vols respostes obertes, configura-hi una clau de l'API d'Anthropic; llavors la Guia consulta Claude enviant la pregunta i un resum del model.

Privadesa de la clau
La clau es desa només al teu navegador. En cada consulta amb clau activa, la pregunta i un resum del model s'envien a l'API d'Anthropic. En mode local no surt res del teu equip.

13Resolució de problemes

Si alguna cosa no va
El model no es carrega o triga molt
Els IFC molt grans poden trigar a processar-se, perquè tota la geometria es genera al navegador. Espera l'indicador de càrrega. Si falla, comprova que és un IFC vàlid (IFC2x3 o IFC4) i no un altre format renombrat.
No veig el model després de carregar-lo
Prem F per enquadrar. Si abans havies aïllat o amagat elements, prem X per tornar a mostrar-ho tot.
La geometria del davant em tapa l'interior
Fes servir T (transparència), H per amagar la selecció, o la Caixa de secció per tallar el model.
La Guia amb clau dóna error de connexió
Si has obert l'HTML directament (file://), el navegador pot bloquejar la crida a l'API. Serveix el fitxer per HTTP (per exemple amb python3 -m http.server) i obre'l a http://localhost:8000.
Vull passar les incidències a un altre programa
Exporta les notes en BCF 2.1 des de la pestanya Notes: el fitxer .bcfzip s'obre a Revit, Navisworks o BIMcollab amb la captura i el punt de vista.
