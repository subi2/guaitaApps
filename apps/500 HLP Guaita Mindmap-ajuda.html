01Introducció

Un editor de mapes mentals en una sola pàgina

Guaita Mindmap et permet construir mapes mentals: un node central del qual pengen idees en forma d'arbre, amb branques, agrupacions i fletxes que hi relaciones al teu gust. És un únic fitxer HTML que s'obre a qualsevol navegador modern.

Pots donar estil als nodes amb temes de color, afegir-hi caselles de tasca, notes, enllaços, imatges i emoticones, i organitzar diferents mapes en pestanyes dins d'un mateix projecte.

Estructura d'arbre

Arrels, fills i germans amb un clic o una tecla.

Multipàgina

Diversos mapes en pestanyes dins un projecte.

Temes i format

Colors, mida de lletra, negreta i alineació.

Exportació rica

PNG, SVG, JSON, HTML, Word i FreeMind.

02Com començar

Del node en blanc al primer mapa
  1. Posa nom al projecte. A dalt a l'esquerra hi ha un camp de text amb «El Meu Projecte»; canvia'l pel nom que vulguis.
  2. Crea el primer node. Fes servir Arrel de la barra d'eines per posar un node nou al llenç, o dóna un títol al mapa amb Títol.
  3. Escriu-hi. Fes doble clic a un node per editar-ne el text; prem Enter per confirmar (o Ctrl+Enter per fer un salt de línia dins el node).
  4. Fes créixer l'arbre. Amb un node seleccionat, afegeix un Fill (o Tab) o un Germà (o Enter).
Desar la feina
Per guardar el mapa, exporta'l a Projecte Complet JSON (vegeu Importar i exportar) i torna'l a importar més endavant per continuar editant.

03La interfície

On és cada cosa
A dalt a l'esquerra: nom del projecte + pestanyes · A dalt a la dreta: Cercar · Desfer/Refer · Importar · Exportar
Barra d'eines lateralSelecció, Títol, Arrel, Fill, Germà, Fletxa, Clau, Núvol, alineació, Tasca, Imatge, Nota, Enllaç, i el panell de Colors/Temes i format de text.
Llenç centralEl mapa. Arrossega el fons per moure't; roda del ratolí o botons de zoom per apropar/allunyar.
A baix a la dreta: controls de zoom (apropar, centrar el mapa, allunyar)

Al llenç pots moure't arrossegant una zona buida i fer zoom amb la roda del ratolí. Els botons de la cantonada permeten apropar, centrar el mapa a la vista i allunyar.

04Crear i editar nodes

La barra d'eines i el ratolí

Amb l'eina Selecció activa pots triar nodes fent-hi clic; arrossega sobre el fons per seleccionar-ne diversos amb un rectangle, o mantén Ctrl mentre cliques per afegir-los a la selecció.

EinaQuè fa
TítolPosa (o treu) el títol principal del mapa.
ArrelCrea un node arrel nou, independent.
FillAfegeix un node fill al node seleccionat (també amb Tab).
GermàAfegeix un node germà al mateix nivell (també amb Enter).
SuprimirEsborra els nodes seleccionats (també amb Supr).

Per editar el text, fes doble clic al node i escriu. Per moure un node, arrossega'l. Pots plegar i desplegar les branques d'un node per amagar-ne els fills.

Alinear diversos nodes
Amb diversos nodes seleccionats, els botons d'alineació els alineen tots a l'esquerra o a la dreta.

05Connexions i agrupacions

Relaciona i emmarca idees

A banda de l'arbre pare-fill, pots crear relacions i contenidors visuals:

EinaQuè fa
FletxaActiva el mode enllaç per traçar una fletxa d'un node a un altre i marcar una relació lliure.
ClauAgrupa els nodes seleccionats amb una clau (accolada), útil per a llistes de subelements.
NúvolEncercla els nodes seleccionats amb un contorn (frontera) que els emmarca com un conjunt.

06Enriquir els nodes

Tasques, notes, enllaços, imatges i emoticones
  • Tasca — afegeix o treu una casella de verificació al node, per fer-ne llistes de coses a fer.
  • Nota — obre una finestra per escriure informació addicional lligada al node.
  • Enllaç — associa una adreça web (https://…) al node; després pots visitar-la.
  • Imatge — puja una imatge al node des del disc, o enganxa-la directament amb Ctrl+V.
  • Emoticones — mentre edites el text, tens un panell d'emojis per categories (cares, símbols, objectes, natura) per inserir-los al node.
Enganxar imatges
No cal passar pel botó Imatge: si tens una imatge al porta-retalls, prem Ctrl+V i s'afegeix directament.

07Estil i format

Colors i tipografia dels nodes

El panell Colors / Temes de la barra lateral ofereix una paleta de temes (Ocean, Forest, Sunset, Berry, Royal, Minimal, Crema, Plom, Núvol, Pedra, Sàlvia, Moca, Rosa Antic, Marí, Argila). Fes clic a un tema per aplicar-lo als nodes.

A Format Text pots ajustar la mida de la lletra (A− / A+), posar negreta o cursiva, i alinear el text del node a l'esquerra, al centre o a la dreta.

08Pestanyes i navegació

Diversos mapes en un projecte

A dalt a l'esquerra, sota el nom del projecte, hi ha les pestanyes. Cada pestanya és un mapa independent dins del mateix projecte. Amb el botó + n'afegeixes una de nova, i fent doble clic al nom d'una pestanya la pots reanomenar.

Per cercar dins el mapa, prem Ctrl+F o obre el quadre de cerca; escriu-hi el text i s'hi ressalten les coincidències. Tanca la cerca amb Esc o la creu.

Centrar la vista
Si t'has perdut pel llenç, el botó de centrar el mapa a la vista (entre els controls de zoom) enquadra tot el contingut de la pestanya activa.

09Importar i exportar

Treu i torna a portar la teva feina

El botó Importar obre fitxers .json, .html i .mm (FreeMind) per continuar treballant amb un mapa existent. El menú Exportar ofereix diversos formats:

FormatPer a què serveix
Projecte Complet JSONArxiu per desar-ho tot i seguir editant més endavant. És el format recomanat per guardar la feina.
Editor Interactiu (.html)Una pàgina web autoeditable amb el mapa incrustat.
Visor Interactiu (.html)Una pàgina per compartir el mapa en mode només lectura.
Esquema Word (.doc)L'estructura del mapa com a document formatat.
Imatge (.png)Imatge de la pestanya activa.
Vector (.svg)Gràfic vectorial de la pestanya activa.
FreeMind (.mm)La pestanya activa en el format del programa FreeMind.
Recomanació
Exporta sovint a Projecte Complet JSON: és l'únic format que conserva tot el projecte (totes les pestanyes, notes, enllaços i estils) per reprendre l'edició sense pèrdues.

11Resolució de problemes

Si alguna cosa no va
He tancat el navegador i he perdut el mapa
Per conservar la feina cal exportar-la explícitament. Fes servir Exportar → Projecte Complet JSON i, quan tornis, Importa aquest fitxer per continuar exactament on eres.
«No pots enganxar fills al títol principal»
El node de títol del mapa no admet nodes fills penjant directament. Enganxa'ls sobre un node normal (arrel o fill).
Tab o Enter no creen nodes
Aquestes dreceres actuen sobre el node seleccionat quan no estàs editant text ni tens cap finestra oberta. Assegura't d'haver seleccionat un node i d'haver tancat qualsevol quadre de nota, enllaç o pestanya.
El mode Visor no em deixa editar
És el comportament esperat: el Visor Interactiu (.html) és de només lectura, pensat per compartir. Per editar, fes servir l'Editor Interactiu (.html) o importa el JSON del projecte.
La imatge enganxada no apareix
Selecciona primer un node i comprova que el porta-retalls conté realment una imatge. Com a alternativa, fes servir el botó Imatge per pujar-la des del disc.
