Federa models

Arrossega un o més .ifc al visor (arquitectura, estructura CYPE, instal·lacions…). Cada model conserva les seves coordenades reals i apareix a la pestanya Models amb ull, color, opacitat, «Color pla», «Aïlla» i «Substitueix…».

Referències acceptades: Rhino .3dm, .obj/.stl/.glb, plànols .dxf (underlay vectorial), fulls .pdf/.png/.jpg a escala arquitectònica (planta o alçats, fons transparent) i núvols de punts .e57/.xyz/.pts.

Alinea

Alinea… a la targeta del model: Manual (desplaçaments, gir, inclinacions, escala — en viu), Georef IFC (llegeix IfcMapConversion/IfcSite; el primer model georeferenciat fixa l'origen) o Punt a punt (clica origen i destí, també sobre núvols de punts).

Secciona per model

La caixa de secció té selector de model: cada model recorda la seva caixa (gir + tres dobles corredisses) i poden estar actives alhora. Seccionar un PDF retalla el full.

Desa la federació

Fitxer → Desa la federació… (Ctrl+S) genera un .federa.json amb origen, càmera, notes BCF i, per model, hash i delta d'alineació. En reobrir-lo, arrossega els mateixos fitxers (posició exacta) o versions noves amb el mateix nom (conserva les correccions manuals).

Normes EIR/BEP

Fitxer → Carrega normes BEP/EIR… accepta .json i .md amb la convenció de taules (vegeu la plantilla BEP-plantilla.md): ## fita + taula ID | Descripció | Filtre | Comprovació | Llindar. L'auditoria (Eines → Auditoria) verifica el model federat contra les normes; el camp Model als filtres acota requisits per disciplina. Clica el xip de la norma per llegir el document.
