· Descobreix la suite d'eines autònomes de SBS BIM Consulting per a arquitectura, BIM i documentació ·

Visualitza, converteix i documenta els teus fitxers al navegador.

Guaita és una col·lecció d'eines autònomes: cada aplicació és un únic fitxer HTML que s'obre a qualsevol navegador modern. Els teus fitxers no s'envien enlloc — tot el processament passa al teu equip, sense instal·lar res ni pujar cap document a cap servidor.

01Fitxers i models BIM

Explora, audita i transforma els teus fitxers de Revit.

Guaita FitxersRevit · CAD

Navegador de fitxers Revit i CAD que llegeix versió, autoria, miniatures i metadades directament del binari, i permet copiar, moure, reanomenar i esborrar amb un pont cap a Revit.

RevitmetadadesminiaturesChromium

Guaita RVT conveRTERVT ↔ RTE

Converteix entre projecte Revit (.rvt) i plantilla (.rte) detectant el tipus de fitxer pel seu Last Save Path intern, sense necessitat d'obrir Revit.

RVTRTEplantilla

02Visors CAD i BIM

Obre i inspecciona dibuixos i models sense el programa original.

Guaita DWFxvisor DWFx

Descomprimeix el document DWFx, converteix les pàgines XAML a gràfics vectorials SVG i les mostra amb capes, propietats, mesures i notes exportables.

DWFxvectorialSVGnotes

Guaita DXFvisor DXF 2D/3D

Visor de fitxers DXF amb capes, tipus d'entitats i blocs, navegació amb NavCube, eina de mesura, vistes i notes desades, i exportació a CSV/HTML.

DXFCAD2D/3Dcapes

Guaita Queryvisor IFC 3D

Visor IFC 3D (web-ifc + three.js): estructura i classes, propietats i quantitats, seccions, mesura, punts de vista, notes BCF, auditoria i comparació de versions.

IFCBIMBCF3D

Guaita IFC Classesfitxa tècnica per classes

Genera fitxes tècniques per classe d'element (portes, finestres, murs, sales...) a partir de l'export ODBC/Excel del Revit o directament d'un fitxer IFC, amb propietats i quantitats tabulades.

IFCExcelODBCfitxa tècnica

03Conversors de PDF

Treu el contingut d'un PDF cap a formats reutilitzables.

Guaita PDF → DXFPDF vectorial → DXF

Converteix un PDF vectorial a DXF: escala i unitats, calibratge amb dos punts, corbes Bézier i filtre de soroll, amb estadístiques de les entitats generades.

PDFDXFvectorialescala

Guaita PDF → .mdPDF → Markdown

Extreu el text d'un PDF i el converteix a Markdown amb PDF.js: arrossega el fitxer, ajusta els paràmetres d'extracció, previsualitza i descarrega. No fa OCR de PDF escanejats.

PDFMarkdownPDF.js

04Markdown

Llegeix i edita documents Markdown amb comoditat.

Guaita .mdvisor Markdown

Visor de Markdown amb pestanyes múltiples, esquema/índex del document, vista de codi dividida, tema clar/fosc i impressió formatada o del codi font.

Markdownvisorpestanyes

Guaita .md editoreditor Markdown

Editor de Markdown complet: edició dual sincronitzada (format + codi), taules i importació CSV/TSV, cerca i substitueix, i exportació a HTML, LaTeX, TXT i PDF.
