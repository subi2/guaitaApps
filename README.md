# Guaita Apps

**Suite d'eines autònomes per a arquitectura, BIM i documentació**

## 🎯 Descripció
Guaita Apps és una col·lecció d'aplicacions HTML autònomes dissenyades per a professionals de l'arquitectura, BIM i documentació.

## 📦 Aplicacions Principals
- **GuaitaFitxers** - Gestor de fitxers
- **Guaita DWG/DXF/DWFX** - Conversors de formats
- **Guaita PDF ConvDXF** - Conversor PDF a DXF
- **Guaita Query** - Eina de consultes
- **Guaita PDF a Markdown** - Conversor PDF
- **Guaita Markdown Editor** - Editor Markdown
- **Guaita RVT ConveRTE** - Conversor Revit
- **Guaita IFC-Classes** - Gestor IFC
- **Guaita Dossier COAC** - Generador dossiers
- **Guaita Historial** - Gestor historial
- **Guaita Mindmap** - Editor mapes mentals
- **Super Editor PDF** - Editor PDF avançat
- **VoltSim Simulator** - Simulador elèctric
- **Zona Climàtica** - Calculadora climàtica

## 🚀 Com Usar
1. Descàrrega els fitxers del repositori
2. Executa l'aplicació desitjada amb el launcher
3. Cada aplicació funciona de manera autònoma

## 📋 Requisits
- Navegador web modern
- JavaScript habilitat
